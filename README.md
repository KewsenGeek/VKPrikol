# Инструкция к программе VKPrikol.

Для того, чтобы начать пользоваться программой, нужно установить все необходимые библиотеки:

`pip3 install -r requirements.txt`

Чтобы запустить программу, нужно ввести в консоль:

`python3 main.py`

Для использования программы используется файл config.ini, в котором нужно указать все нужные параметры.

* login и password - это логин и пароль от вашей страницы ВКонтакте соответственно.
* group_id и album_id - это ID группы и альбома соответственно. Как получить эти два значения смотрите ниже.
* UploadLimit - это сколько кругов будут загружаться фотографии, то есть при значении 2 загрузятся 2 папки с файлами.
* * При значении 0 фото будут загружаться до тех пор, пока вы не остановите программу.
* FolderWithPhotos - это название папки с фотографиями. Внимание! Папка должна быть в одной папке с программой!
* AutoCheckForUpdates - автообновление (пока в разработке).
* Version - версия программы.
* PythonVersion - версия python, на котором была написана программа.

# Получение ID альбома и группы
Чтобы получить ID альбома, нужно перейти в альбом, и в ссылке выделить цифры после нижнего подчеркивания. Это и будет ID альбома.
Чтобы получить ID группы, нужно перейти на [сайт](https://regvk.com/id/) и ввести туда ссылку на группу. Это и будет ID группы.
